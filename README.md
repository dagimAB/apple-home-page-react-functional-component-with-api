Apple Home Page (React Functional Component)
A faithful recreation of Apple's home page built with React functional components using api key for fetching youtube video — focused on layout, responsiveness, and modern CSS. This project is an educational reimplementation and is not affiliated with or endorsed by Apple Inc.


Features
Built with React using only functional components and hooks
Responsive layout that adapts to desktop, tablet, and mobile
Clean CSS structure with reusable styles
Semantic HTML for accessibility
Lightweight, no external UI frameworks
Built with
JavaScript (React)
HTML5
CSS3 (Flexbox / Grid)
Getting started
Prerequisites

Node.js (>=14) and npm or Yarn
Install

Clone the repo git clone https://github.com/dagimAB/apple-home-page-with-react-functional-component.git
Install dependencies cd apple-home-page-with-react-functional-component npm install
or
yarn
Run locally

Start the dev server: npm run dev

Create a production build: npm run build

The build output in /build can be deployed to GitHub Pages, Netlify, Vercel, etc.
Example for GitHub Pages: use gh-pages or a static hosting setup.
